Detailed Description

Base class for all classes.

This is the base class whose all others classes must inherit otherwise no call to new() will be possible due to the type of the new() first argument Since the notion of class is implemented through structures and compile time defined description variables, fields in Class are related to the downmost class in the inheritance tree.
