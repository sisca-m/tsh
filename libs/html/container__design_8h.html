typedef Object*(* t_sub) (Object *self, Class *type, int begin, int len)

Function pointer which represents a member function returning a subobject from an Object.

Parameters
selfIt's an Object * to avoid warnings when doing implicits upcastings.
typeA description class variable or an upcasted object. For containers and in the case of an upcasted object, the already existants contained values are not deleted due to the fact that the conversion use push_back().
beginAn index which points to the beginning of the subobject. A negative index means the substring start at n positions from the end.
t_String *string = new(_string, "filename.c", 0);
string->substr(string, -2, 2); // Returns ".c"
string->substr(string, 3, 4); // Returns "enam"
lenThe length on which the subobject is extract.
Returns
A memory allocated Object containing the subobject.
Warning
A negative begin which is in absolute greater than the size of the Object will leads you to a crash.
Note
If the len is greater than the size of the strings no crash will happen.
