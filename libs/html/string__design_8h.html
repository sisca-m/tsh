typedef Object*(* t_str_split) (Object *self, Class *type, char *sep)

Function pointer which represents a member function returning an Object instancied from t_container or one of its derived class. The Object returned contains the value contained in Object self but splitted with separators. The returning Object has to be - or derived of - a t_container to be able to store severals values in case where at least one separator is found in self.

Parameters
selfIt's an Object * to avoid warnings when doing implicits upcastings. Very usually it's a string.
typeA container class description variable or a container. The behaviour is defined by the implementation of push_back().
sepEach char in sep is a separator.
Returns
The newly created Object which store the splitted value of self.
