Instanciate a {\ttfamily \hyperlink{structClass}{Class}} 

Create a memory allocated object from {\ttfamily class} by copying the value of {\ttfamily class} and then call the constructor of the object. Each classes have a description variable to define the functions member and attributes of the class. Those variables are usually pass to new but you can give any object you want if you know what you are doing. \hyperlink{bool_8h_ab719d996d8c101c34248bc256b8dcfd8}{t\+\_\+bool}
